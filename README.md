# DeliveryCheckService
택배 조회 구현 서비스


참조 : https://tracker.delivery/guide/

택배 api를 통해 택배조회 서비스 
현재는 대한통운, 한진택배, 우체국택배 택배조회 구현
타 택배조회는 기존에 구현된 코드를 통해 쉽게 구현 가능

현재는 Logcat을 통해 시간대별 위치를 나타내줌
